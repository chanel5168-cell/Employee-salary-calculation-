# 員工薪資計算系統 (LINE LIFF)

專為小型公司（< 20 人）設計的薪資計算網頁，可嵌入 LINE 供員工使用，支援月薪、論件計酬、計時計酬三種薪資模式，並可匯出多工作表的 Excel 報表。

## 快速開始

直接用瀏覽器開啟 `index.html` 即可使用，資料儲存在瀏覽器本機 (localStorage)。

如要讓多人使用或嵌入 LINE，請參考 `DEPLOY.md`。

## 主要功能

**員工管理**：新增、編輯、刪除員工資料（姓名、職稱、月薪、時薪）。

**每日記錄**：支援兩種計酬方式，可每天輸入。
- 論件計酬：品名、數量、單價 → 自動計算小計
- 計時計酬：起訖時間 + 休息時間 → 自動計算實際工作時數 × 時薪
- 當日合計與整月加總自動統計

**月薪計算**：底薪、加班費（平日 ×1.34 / 假日 ×1.67）、獎金、請假扣款、勞健保、其他扣款，自動計算應發與實發。

**Excel 匯出**：一鍵產生多工作表 Excel（月薪摘要/月薪明細/每日明細/每日摘要/員工月總）。

**資料備份**：可匯出 JSON 備份、還原、清除。

## 技術規格

- 單一 HTML 檔案（HTML + CSS + JavaScript 三合一）
- 使用 LINE LIFF SDK（v2）整合 LINE
- 使用 SheetJS (xlsx) 產生 Excel
- 繁體中文介面
- 響應式設計 (手機/平板/電腦)
- 資料儲存於瀏覽器 localStorage
