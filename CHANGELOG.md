# 版本歷程

## v1.0.0 (2026-04-20)

### 初版功能
- 員工管理（新增/編輯/刪除）
- 月薪計算（底薪、加班、獎金、勞健保、扣款）
- LINE LIFF SDK 整合
- Excel 匯出（摘要 + 明細雙工作表）
- JSON 備份與還原

## v1.1.0 (2026-04-20)

### 新增每日記錄
- 新增「每日」頁籤
- 論件計酬：品名、數量、單價自動計算小計（品名會記憶過往輸入供下拉建議）
- 計時計酬：起訖時間自動計算時數（支援跨日）
- 當日合計顯示
- 整月加總表格（按日期列出論件/計時金額與當日合計）
- Excel 匯出新增三個工作表：每日明細 / 每日摘要 / 員工月總

## v1.2.0 (2026-04-20)

### 改善休息時間扣除
- 休息時間新增快速按鈕（無休息/30分/1小時/1.5小時/2小時）
- 自動換算休息分鐘等於幾小時
- 計算明細框清楚顯示：時段總長 − 休息時間 = 實際工作時數
- 小計欄位顯示完整算式（例：NT$ 1,520 (8.00h × 190)）
- 提取 `calcSpanMins` 函式分離總時段與工作時數計算
