如何佈署到 LINE？

1. 將此 HTML 上傳到 HTTPS 主機 (如 GitHub Pages / Netlify / Vercel)
2. 前往 developers.line.biz 登入並建立 Provider
3. 建立「LINE Login」Channel，在 LIFF 分頁新增 LIFF App
4. Endpoint URL 填入您的 HTML 網址，Size 選 Full
5. 複製 LIFF ID 貼到上方欄位並儲存
6. 將 LIFF URL (https://liff.line.me/<LIFF_ID>) 分享到您的群組
